# reagent-mvsa
Model-View-Signal-Action pattern implementation for ClojureScript using Reagent.

WIP.

## Requirements

ClojureScript 1.7.170 or higher.

## Installation
Add this to your dependencies in project.clj:

```clj
[reagent-mvsa "0.1.0"]
```

See examples/ folder.
